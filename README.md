## Installation

```bash
# Install dependencies
npm install
```

## Running the App

### Development Mode

```bash
# Start the development server
npm run dev
```

The app will be available at `http://localhost:3000`.

### Build for Production

```bash
# Build the project for production
npm run build
```

### Run Production Version

```bash
# Serve the production build
npm start
```
